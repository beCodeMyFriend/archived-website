Requerimiento:

Configurar el servicio SNMPD en CentOS para poderlo graficar desde un Cacti remotamente.

Solución:

Paso 1. Instalar SNMPD.

yum install net-snmp-utils

Paso 2. Configurar SNMPD.

mv /etc/snmp/snmpd.conf /etc/snmp/snmpd.conf.original
nano -w /etc/snmp/snmpd.conf

rocommunity public
syslocation "TuServidor DataCenter"
syscontact none@none.com

Paso 3. Reiniciar servicio.

/etc/init.d/snmpd restart

Paso 4. Probarlo.

Desde el servidor que tiene el Cacti instalado, podéis ejecutar el siguiente comando y ver que la salida no de error alguno.

snmpwalk -v 2c -c public IP_HOST_CON_SNMPD system
