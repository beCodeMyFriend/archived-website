Esto no es más que un mini-tutorial para que otros eviten perder la media hora (o más) que yo he perdido intentando hacer funcionar esto. No sé si es cosa de mi configuración actual o de la configuración por defecto de Snow Leopard, pero he aquí mi problema.

Al instalar el módulo usando PECL con:

> sudo pecl install oauth

Y reiniciar el servidor Apache:

> sudo apachectl restart

Apache no carga la extensión. Tras revisar el error_log vemos que:

PHP Warning: PHP Startup: Unable to load dynamic library '/usr/lib/php/extensions/no-debug-non-zts-20090626/oauth.so' - dlopen(/usr/lib/php/extensions/no-debug-non-zts-20090626/oauth.so, 9): image not found in Unknown on line 0

Resulta que lo instala en un directorio distinto al que apache busca las extensiones PHP. Por tanto copiamos de uno al otro (en mi caso):

> sudo cp /opt/local/lib/php/extensions/no-debug-non-zts-20090626/oauth.so /usr/lib/php/extensions/no-debug-non-zts-20090626/
> sudo apachectl restart

Podemos comprobar que la extensión está cargada lanzando un phpinfo().
